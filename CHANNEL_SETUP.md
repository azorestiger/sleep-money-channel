## 🎬 Video Setup & Upload

### Upload Your First Videos

1. **Upload first 10 videos before launch**
   - Ensures consistent content for new subscribers
   - Allows time to optimize titles/descriptions

2. **Video Requirements:**
   - Format: MP4 (H.264 video, AAC audio)
   - Resolution: 1920x1080 (Full HD)
   - Frame rate: 24-30 fps
   - Duration: 480 minutes (8 hours typical)
   - File size: 5-25 GB (YouTube accepts)

3. **Upload Settings:**
   - Privacy: **Public**
   - Allow embedding: **Yes**
   - Show more options → Enable comments, likes, all features
   - Category: **People & Blogs**
   - Language: **English**
   - Caption certification: English captions included

### Create Thumbnails

**Specifications:**
- Dimensions: 1280 x 720 pixels
- Format: JPG or PNG
- File size: < 2 MB
- Safe area: Keep important elements within center 60% (mobile view)

**Design Formula:**
1. **Background:** Calming image (moon/stars/night sky)
2. **Overlay:** Semi-transparent dark box (with text)
3. **Text:** Bold, white/gold, 44pt+ font
   - Example: "💰 MONEY SLEEP"
4. **Additional:** Add subtle prosperity symbols
5. **Avoid:** Too much text, unclear images, poor contrast

**Tools:**
- Canva (easiest, free plan available)
- Photoshop
- GIMP (free)
- Template: Use gold/purple/white color scheme

## 📝 SEO & Metadata Optimization

### Title Formula
**Template:** `[LENGTH] Sleep Affirmations for [THEME] | [BENEFIT]`

**Examples:**
- "8-Hour Sleep Affirmations for Money | Financial Abundance Manifestation"
- "Sleep Meditation - Attract Wealth While You Sleep | Money Affirmations"
- "Subconscious Reprogramming | Sleep Affirmations for Prosperity & Abundance"

**Best Practices:**
- Include video length (8 hour, 10 hour, etc.)
- Include main benefit (Money, Abundance, Freedom)
- Add secondary keyword (Sleep, Meditation, Affirmations)
- Keep under 60 characters ideally
- Use | as separator for clarity

### Description Formula
**Structure:** (use template from `config/channel_setup.json`)

1. **Hook** (first 2 lines - appears before "show more")
   - "🌙 8-HOUR SLEEP MONEY AFFIRMATIONS 💰"

2. **Benefits list** (what viewers get)
   - Calming affirmations
   - Wealth manifestation
   - Subconscious reprogramming

3. **How to use** (step-by-step)
   - Use headphones/speakers
   - Play on loop throughout night
   - Listen for 21 days

4. **Affirmations preview** (top 5-6 affirmations)

5. **CTA** (Call to action)
   - Subscribe & turn on notifications
   - Like for algorithm
   - Share to help others

6. **Hashtags** (8-15 relevant)
   - #SleepAffirmations #MoneyAffirmations #Manifestation

7. **Resource links** (affiliate)
   - Meditation app recommendations
   - Sleep tech suggestions

## 📊 Analytics to Track

### YouTube Studio Metrics
- **Watch time** (target: 30+ min per viewer)
- **Click-through rate** (target: 4%+)
- **Average view duration** (target: >50% of video)
- **Retention graph** (where viewers drop off)
- **Audience demographics** (age, gender, location)
- **Traffic sources** (YouTube search, browse, external, etc.)

### Success Indicators
| Metric | Target | Note |
|--------|--------|------|
| Avg view duration | 30+ min | Long videos = high retention |
| Subscriber growth | 10%+ monthly | Sustainable growth |
| Click-through (description) | 4%+ | Your SEO & titles work |
| Playlist adds | High | Videos match search intent |
| Shares | Growing | Content resonates |

## 💡 Optimization Tips

### If Views Are Low
- Review thumbnail vs. competitors
- Check title relevance to search
- Improve description SEO
- Post Shorts regularly (drives traffic)

### If Watch Time Is Low
- Check retention graph (where do people leave?)
- Make first 30 seconds more engaging
- Add pauses/pacing if too fast
- Shorten if consistently watched <50%

### If No Engagement
- Create better calls-to-action
- Pin comment asking questions
- Create community posts asking for feedback
- Engage actively in comments (reply to 100%)

### To Grow Faster
- Post Shorts 5-7x per week
- Create trending topic affirmations
- Collaborate with similar channels
- Use trending sounds in Shorts
- Upload consistently (train algorithm)
